## SETUP

Clone this repo from your home directory (~):

1. Run `./dotfiles/initiate.sh` 
2. Install Vundle: `git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle`
3. Launch vim and run `:PluginInstall`
